# FPGA Digital Systems Portfolio – Dawei Cai

Electrical & Computer Engineering student at the University of Washington (UW Seattle)

This repository documents hardware systems I designed and implemented on the Intel DE1-SoC FPGA board using SystemVerilog.
All designs were simulated in ModelSim and verified on physical hardware.

---

## Skills Demonstrated

* SystemVerilog RTL Design
* Finite State Machine (FSM) & Datapath Architecture
* Synchronous Memory Interfaces
* Hardware Debugging & Verification
* ModelSim Testbench Development
* Quartus FPGA Synthesis & Implementation

---

## Implemented Hardware Systems

### Memory Systems

* 32×4 RAM with read/write control
* Dual-port memory interface
* FIFO buffer with empty/full detection

### Algorithmic Hardware

* Binary search hardware accelerator
* Bit-counting circuit using ASMD design

### Graphics

* VGA framebuffer controller
* Bresenham line drawing engine

### Audio Processing

* Real-time microphone loopback
* ROM-based tone generator
